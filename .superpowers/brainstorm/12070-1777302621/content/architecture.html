🛡️ Sentinel re-verifies (on schedule)
Walks rows where verify_status='pending' OR last_checked stale.
Re-runs verify_signal. Updates verify_status.
Catches drift (rollback, file deleted, deploy reverted).
💓 Liveness watchdog (Hybrid A+C)
6a · External heartbeats: local-agent-server.js + launchd jobs touch <agent>.heartbeat every N min. Process-layer write — NOT from inside agent prompt. Catches process death.

6b · Dispatch-stale watch: any row in_flight > threshold → flagged wedged. Catches prompt pinning (today's sentinel).
⏱️ Bash hard timeout (defense-in-depth)
SIGTERM any spawned child > 10–30 min. Prevents until X; sleep wedges.
🏷️ Banner renderer fix
Compute current mtime age at render time. No more "3.0h" frozen for 28h.
